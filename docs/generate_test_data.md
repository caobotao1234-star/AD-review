## 使用说明

1. 每组开 3 个独立对话（1个JSON + 2个图片）
2. JSON 对话直接复制粘贴提示词，GPT 会输出 JSON 文本，你保存为对应文件名
3. 图片对话用 GPT-4 的图片生成功能，生成后下载保存为对应文件名
4. 所有文件放入项目的 `input/` 目录：
   ```
   input/
   ├── case_01.png
   ├── case_01.json
   ├── case_02.png
   ├── case_02.json
   ├── ...
   └── case_10.json
   ```
5. 图片格式 png/jpg/jpeg/webp 都可以
